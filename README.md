# 📚 Libra-Track

A Python-based Library Management System built with Flask to track books, manage member records, and handle issue/return operations.

## Features
- Add, view, and manage books
- Register and manage members (Students & Faculty)
- Issue and return books with automatic fine calculation
- Overdue reminders using background threading
- Data persistence using JSON storage

## Tech Stack
- Python
- Flask
- HTML/CSS (Jinja templates)
